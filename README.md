Chocolate Sales Analysis Report:
This report provides a consolidated and visual summary of chocolate sales data using Excel Pivot Tables. It is designed to help stakeholders quickly understand sales patterns, product performance, top-selling categories, and key business insights.

| Feature                         | Purpose                                                      |
| ------------------------------- | ------------------------------------------------------------ |
| **Pivot Tables**                | Summarizing and grouping large dataset values.               |
| **Pivot Charts**                | Visual trend analysis and comparison.                        |
| **Calculated Fields (if used)** | Creating custom metrics for deeper insight.                  |
| **Formatting & Filters**        | Enhancing readability and facilitating interactive analysis. |
